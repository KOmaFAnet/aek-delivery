# aek-delivery
AEK Delivery is a frontend project for a fictional food delivery service designed for restaurants and food businesses.

## 🚀 Features

- Responsive multi-page website
- Category-based pages (Fast Food, Sushi, Confectionery, Catering)
- Interactive navigation with dropdown menu
- Mobile sidebar menu
- Request form for business inquiries
- Custom UI with cards and gradients

## 🧠 Concept

This project is focused on a **B2B delivery service**, where restaurants can connect to a delivery network instead of customers ordering food.

Each category page highlights specific advantages for different types of food businesses.

## 🛠 Technologies

- HTML
- CSS (Flexbox, responsive design)
- JavaScript (DOM manipulation)
